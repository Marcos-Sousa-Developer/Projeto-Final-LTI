Resumo das solicitações feitas durante um teste de carga usando o Locust. 

Tipo: o método HTTP usado para as solicitações (por exemplo, GET, POST, etc.).
Nome: o caminho da URL ou endpoint que foi acessado.
Nº de pedidos: o número total de pedidos feitas para esse endpoint específico durante o teste de carga.
Falhas: O número de nº de pedios com falha para esse endpoint.
Mediana (ms): o tempo médio de resposta (em milissegundos) para as solicitações feitas a esse endpoint.
90%ile (ms): O tempo de resposta abaixo do qual 90% das solicitações caem (em milissegundos).
99%ile (ms): O tempo de resposta abaixo do qual 99% das solicitações caem (em milissegundos).
Average (ms): O tempo médio de resposta (em milissegundos) para as solicitações feitas para esse endpoint.
Min (ms): O tempo mínimo de resposta (em milissegundos) registrado para esse endpoint.
Max (ms): O tempo máximo de resposta (em milissegundos) registrado para esse terminal.
Tamanho médio (bytes): o tamanho médio da resposta (em bytes) recebida para esse terminal.
RPS atual: o número atual de solicitações por segundo feitas para esse ponto de extremidade.
Falhas/s atuais: o número atual de solicitações com falha por segundo para esse endpoint.
